# Aura Score — Free Personal Aura Reading & Energy Test

Discover your personal aura colors and energy reading with a free aura score test. Reveal your unique aura meaning, color, and personality traits in seconds.

**Live site: [https://aura.csskey.com](https://aura.csskey.com)**

## Features

- **Aura color quiz** — Find your dominant aura color
- **Energy reading** — Get a personalized energy score
- **Aura meaning guide** — Learn what each aura color represents
- **Personality insights** — See how your aura reflects your traits
- **Shareable results** — Download or share your aura profile

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **SEO:** react-helmet-async, JSON-LD Schema
- **Hosting:** Cloudflare Workers

## Quick Start

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```
